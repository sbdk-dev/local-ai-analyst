# Claude-Analyst Examples

This directory contains example scripts demonstrating how to use Claude-Analyst programmatically.

## Running Examples

```bash
# From the repository root
python examples/basic_query.py
```

## Available Examples

### 1. basic_query.py
Demonstrates basic semantic layer queries:
- Getting total user count
- Grouping by dimensions
- Calculating measures
- Limiting results

**Usage**:
```bash
python examples/basic_query.py
```
